#  SQL Analysis - Superstore Dataset

##  Overview
This folder contains all SQL queries used to analyze the Superstore dataset.  
The objective is to extract meaningful business insights related to **sales, profit, customers, and regional performance**.

---

##  Tools Used
- SQL (MySQL)
- Dataset preprocessed using Python (Pandas)

---

##  Files
- `superstore_analysis.sql` → Contains all SQL queries used in the project

---

##  Business Questions Covered

###  Category & Sub-Category Analysis
- Which category generates the highest profit?
- Which sub-category is the most and least profitable?
- Which sub-categories are causing losses?

###  Regional Analysis
- What is total sales and profit by region?
- Which state contributes the most to overall profit?
- Which states have high sales but low profit?

###  Product Analysis
- Which products are consistently loss-making?
- Which are the top 10 loss-making orders?

###  Customer Analysis
- Who are the top 5 customers by profit?

###  Time-Based Analysis
- What is the monthly sales and profit trend?
- What is the year-over-year growth in sales?

###  Profitability & Discount Analysis
- How does discount impact profit?
- Which sub-categories have the lowest profit margin?
- Rank sub-categories based on profit within each category

---

##  Key Insights

- High discounts negatively impact profit and often lead to losses  
- Sales do not always translate into profitability  
- A small group of customers contributes significantly to total profit  
- Some sub-categories and products consistently generate losses  
- Certain regions and states show inefficiencies despite high sales  
- Sales and profit exhibit seasonal trends  

---

##  Purpose
This SQL analysis helps:
- Identify **loss-making areas**
- Optimize **pricing and discount strategies**
- Understand **customer contribution**
- Support **data-driven decision making**

---

##  How to Use
1. Import the dataset into your SQL database  
2. Run the queries from `superstore_analysis.sql`  
3. Analyze the results to derive insights
